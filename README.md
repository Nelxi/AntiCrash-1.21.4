# AntiCrash

Защита от серверных эксплойтов для Minecraft 1.21.4

## О моде

Играя на публичных серверах, я не раз сталкивался с ситуацией, когда администрация или игроки с определенными правами могли крашнуть клиент буквально одной командой. Спам частиц, странные пакеты, переполнение памяти — всё это приводило к вылету игры и необходимости перезапуска.

AntiCrash решает эту проблему. Мод работает на стороне клиента и перехватывает потенциально опасные действия сервера до того, как они успеют вызвать краш.

## Что блокируется

- **Спам частиц** — ограничение до 10 000 частиц в секунду
- **Флуд пакетов** — автоматическое ограничение скорости обработки повторяющихся пакетов
- **Переполнение памяти** — принудительная очистка при высокой нагрузке
- **Некорректные пакеты** — фильтрация и логирование подозрительных данных
- **Критические исключения** — перехват краша на уровне потоков

Мод работает прозрачно и не требует настройки. Все заблокированные атаки записываются в лог для анализа.

## Установка

1. Скачай и установи [Fabric Loader](https://fabricmc.net/use/) для версии 1.21.4
2. Скачай [Fabric API](https://modrinth.com/mod/fabric-api)
3. Скачай AntiCrash из раздела [Releases](https://github.com/Nelxi/AntiCrash-1.21.4/releases)
4. Помести оба `.jar` файла в папку `.minecraft/mods`

## Требования

- Minecraft 1.21.4
- Fabric Loader 0.16.0 или новее
- Fabric API
- Java 21
